## 不可破坏边界

最高优先级：

```text
Soul scope = userId + personaId
```

禁止：

- 全局 `DeceasedSoul`
- 只按 `personaId` 查询 Soul
- 跨用户聚合人格
- A 用户纠正影响 B 用户
- A 用户节点记忆进入 B 用户上下文
- 删除 A 用户数据影响 B 用户
- 用户端暴露 `SoulVersion` / `SoulSnapshot` / `SoulUpdateProposal` / `MemoryItem` / `scope` / `retrieval` / `evidence`

当前云端 demo 只是公开演示环境：

- 不输入真实用户隐私。
- 不上传真实逝者资料。
- 不接生产数据库。
- 不接真实微信 / 支付 / LLM Key。
- 不当正式产品入口传播给真实用户。
